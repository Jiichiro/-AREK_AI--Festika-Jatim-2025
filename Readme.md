
# Festika Jatim 2025

## Deskripsi
Proyek __[AREK_AI] Festika Jatim 2025__ adalah aplikasi untuk mendukung Festival Kesenian Jatim 2025. dalam ini berisi informasi terkait tempat tempat wisata di __Mojokerto__

## Instalasi
```bash
git clone https://github.com/Jiichiro/-AREK_AI--Festika-Jatim-2025
cd [AREK_AI]\ Festika\ Jatim\ 2025
```

## Penggunaan
